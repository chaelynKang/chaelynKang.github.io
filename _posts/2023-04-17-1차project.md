```
# 공통 설정

en
conf t
no ip domain lookup
line c 0
logg syn
exec-timeout 0
exit
ali exec c conf t
ali exec r sh run
ali exec i sh ip route
ali exec b sh ip int b 
hostname
```

```
# MAC address & VTP

ISP]
en
conf t
int f1/0
shut
mac-add xxxx.xxxx.xxxx
no shut
ip add dhcp

DSW1 - S_SW]
conf t
vtp mode transparent

ASW1 - SSW]
no ip routing
```
